# 半導体工場シミュレーター (Semi Factory Simulator)

半導体工場の製造ラインを組んで回すシミュレーションゲームのプロトタイプ。
経営・マネタイズではなく「ラインを作って流す」工程そのものを楽しむタイプの
ゲームを目指しています(Factorio / Minecraft工業mod系が好きな人向け)。

## このプロトタイプで検証すること

半導体製造の最大の特徴である **リエントラントフロー(再入工程)** —
ウェハが同じ種類の装置を何度も再訪すること — がゲームとして面白いかどうか。

工程レシピは9ステップで、成膜・露光・エッチングを2周する最小構成の
リエントラントフローになっています:

```
初期洗浄 → 成膜① → 露光① → エッチング① → 成膜② → 露光② → エッチング② → 最終洗浄 → 検査
```

露光装置は処理が最も遅く、かつレシピ中2回通るため、自然にボトルネックになります。

## 遊び方

```bash
npm install
npm run dev
```

ブラウザで表示されたURLを開く。

1. 右のパレットから装置(洗浄・成膜・露光・エッチング・検査)を選び、盤面をクリックして設置
2. 左の投入口からロット(FOUP)が自動投入され、レシピ順に空いている装置へ流れる
3. すべての工程を終えたロットは右の出荷口へ運ばれ、完成数にカウントされる

### メカニクス

- **スループット**: 直近60秒の完成ロット数/分。ラインの太さの指標
- **歩留まり**: 各工程で少しずつ欠陥が乗る。装置は使うほど汚れ(清浄度低下)、
  汚れた装置は欠陥率が上がる。装置を選択して「メンテナンス」で回復(その間停止)
- **廃棄**: 検査時点で歩留まり40%未満のロットはスクラップされる
- **ボトルネック可視化**: 右パネルの「プロセスフロー」に工程ごとの仕掛かり数が出る。
  どこで詰まっているかが一目でわかるので、装置の増設判断に使う

### 操作

| 操作 | 効果 |
|---|---|
| パレット選択 → 盤面クリック | 装置を設置 |
| 盤面の装置をクリック | 選択(情報・メンテ・撤去) |
| 右クリック / Esc | 設置・撤去モード解除 |
| 投入間隔スライダー | ロットの投入ペースを調整 |
| 1x / 2x / 4x | シミュレーション速度 |

## 構成

- `src/config.ts` — 装置定義・工程レシピ・チューニング定数
- `src/sim.ts` — シミュレーション本体(ロット搬送・ディスパッチ・歩留まり)
- `src/render.ts` — Canvas描画
- `src/ui.ts` — サイドパネルUI
- `src/main.ts` — 起動・入力・ゲームループ

## ロードマップ(プロトタイプ検証後)

- 搬送系(天井走行レール / OHT)の敷設をプレイヤーが行う
- 工程レシピをプレイヤーが設計できるレシピエディタ
- 製品の複数種類化とテックツリー(ダイオード → ロジック → DRAM → CPU)
- プロセスノード微細化による工程数の増加
- クリーンルームのゾーニングと汚染の空間シミュレーション
